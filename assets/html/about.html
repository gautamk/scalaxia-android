About Scalaxia

what's Scalaxia BTW ?

Welcome to Scalaxia!!! the galaxy of Scala tweets

This project is inspired from the Groovy community website groovytweets.org.

The idea is to unleash the power of twitter to make hub for the Scala related tweets around the planet.

How it works?

This uses Twitter API to scroll out the tweets and using a basic pattern matching to extract Scala related technology tweets from the friends of @scalaxia twitter handler.
This application list of all the Friends tweets which has Scala relate keywords.

How to make my tweets to be added ?

All you need to do is using your twitter id follow our @scalaxia handler. With in some days your account will be refollowed. Once this happens then any of your twitter having any scala related text will be added automatically into scalaxia website.

How long my tweet will be displayed?

We currently listing max of 30 current tweets displayed in our site.
